# Projeto Final de Compiladores 2/2019

## Linguagem Estanisml

Linguagem template feita em Lark que traduz seu código para HTML.

## Instalação e Execução

Install Docker-compose

### Run

>> make build

>> make run

### Run test

>> python3 -m pytest tests/teste1.py
